# smad1
## Использование
<key1> <key2> <filename>

## Ключи
1. <key1>
..1. --gen-sigm - Генерирует сигму
..1. --gen-points - Генерирует точки
1. <key2>
..1. --error - Добавляет помехи в рассчёты
..2. --no-error - Исключает помехи из рассчётов
1. <filename> - Имя файла, куда будут сохранены результаты генерации
