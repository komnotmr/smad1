# smad1
## Использование
key1 key2 <filename>

## Ключи
1. **key1**
* ***--gen-sigm*** - Генерирует сигму
* ***--gen-points*** - Генерирует точки
1. **key2**
* ***--error*** - Добавляет помехи в рассчёты
* ***--no-error*** - Исключает помехи из рассчётов

**filename** - Имя файла, куда будут сохранены результаты генерации
